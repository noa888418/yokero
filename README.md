# yokero - アクションゲーム

## 遊び方

### 簡単な起動方法
1. **run.bat** をダブルクリックしてゲームを起動します

### 操作方法
- **上下キー**: プレイヤーを上下に移動
- **スペースキー**: ゲーム開始
- **ESCキー**: 一時停止 / メニューに戻る
- **マウスクリック**: ボタン操作

## セットアップ（初回のみ）

### Pythonのインストール
1. [Python公式サイト](https://www.python.org/downloads/)からPythonをダウンロード・インストール
2. インストール時に「Add Python to PATH」にチェックを入れる

### pygameのインストール
コマンドプロンプトまたはPowerShellで以下を実行：
```
python -m pip install pygame
```

または
```
py -m pip install pygame
```

## ゲームの特徴
- 障害物を避けて進むアクションゲーム
- 4種類のアイテムで様々な効果を獲得
- ハイスコアを目指してプレイ
- 時間経過で難易度が上昇

## トラブルシューティング

### pipコマンドが認識されない場合
- Pythonがインストールされているか確認
- `python -m pip` または `py -m pip` を使用してみてください

### ゲームが起動しない場合
- Pythonとpygameが正しくインストールされているか確認
- エラーメッセージを確認してください
